# 견적서 VAT 표시 기준

## 기본 원칙

부가세 미포함으로 나가는 견적서는 없다. 모든 견적서의 견적금액은 **(VAT 포함)** 으로 표시된다.

## 단가 VAT포함 체크 여부에 따른 합계 표시

### 1. VAT포함 체크 O (단가에 VAT가 포함된 경우)

작성한 단가가 이미 VAT가 포함된 금액이므로, 합계 영역에 **합계만** 표시한다.

```
┌──────────┬──────────────┬────────┐
│  합  계  │ ￦ 3,726,000 │ 천단위 │
│          │              │ 절사   │
└──────────┴──────────────┴────────┘
```

- 소계 행: 표시하지 않음
- 부가세 행: 표시하지 않음
- 합계 행: 단가 합계에 절사 적용한 금액

### 2. VAT포함 체크 X (단가에 VAT가 미포함된 경우)

작성한 단가가 VAT 미포함 금액이므로, **소계 → 부가세(10%) → 합계** 3행을 모두 표시한다.

```
┌──────────────┬──────────────┬────────┐
│  소  계      │ ￦ 3,726,000 │        │
├──────────────┼──────────────┤        │
│ 부가세 (10%) │ ￦   372,000 │        │
├──────────────┼──────────────┼────────┤
│  합  계      │ ￦ 4,098,000 │ 천단위 │
│              │              │ 절사   │
└──────────────┴──────────────┴────────┘
```

- 소계 행: 품목 합계에 절사 적용
- 부가세 행: 소계의 10%에 절사 적용
- 합계 행: 소계 + 부가세

## 절사(ROUNDDOWN) 기준

| 설정값 | 비고란 표시 |
|--------|-------------|
| 절사 없음 | (표시 안함) |
| 10 | 십원단위 절사 |
| 100 | 백원단위 절사 |
| 1000 | 천단위 절사 |
| 10000 | 만단위 절사 |

절사 적용: `Math.floor(value / unit) * unit`

절사는 비고(Remarks) 칸에 한글로 표시한다. 합계 라벨에는 절사 정보를 표시하지 않는다.

## 견적금액 한글 표시

견적서 상단의 견적금액은 항상 합계 금액을 한글로 변환하여 표시한다.

예: `일금 삼백칠십이만육천원 정 ( ￦ 3,726,000 ) (VAT 포함)`
