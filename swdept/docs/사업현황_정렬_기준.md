# 사업현황 리스트 정렬 기준

## 적용 위치
- 페이지: SW 사업 추진 현황 리스트 (`/projects/status`)
- 파일: `SwController.java` → `projectList()` 메서드

## 정렬 규칙

| 우선순위 | 컬럼       | 필드명    | 정렬방향         |
|---------|-----------|----------|-----------------|
| 1       | 년도       | year     | 내림차순 (DESC)  |
| 2       | 시도명     | cityNm   | 오름차순 (ASC)   |
| 3       | 시군구명   | distNm   | 오름차순 (ASC)   |
| 4       | 시스템명   | sysNm    | 오름차순 (ASC)   |

## 설명
- 최신 년도가 먼저 표시된다.
- 같은 년도 내에서는 시도명 → 시군구명 → 시스템명 순으로 가나다 오름차순 정렬한다.
- 검색(키워드 필터) 시에도 동일한 정렬이 적용된다.

## 적용 코드
```java
Sort sort = Sort.by(Sort.Direction.DESC, "year")
        .and(Sort.by(Sort.Direction.ASC, "cityNm"))
        .and(Sort.by(Sort.Direction.ASC, "distNm"))
        .and(Sort.by(Sort.Direction.ASC, "sysNm"));
```
