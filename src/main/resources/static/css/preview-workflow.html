🤖 자동화 규칙

서버 재시작 — 승인 없이 즉시 (bash server-restart.sh)

"작업완료" 트리거 — confirm 1회 추가 (단발 발화 즉시 push X):

  1. 변경분 echo (status / diff / log preview)
  2. 사용자 confirm 요청
  3. "yes"/"y"/"응"/"ok" → 진행 · 다른 응답 → cancel

저위험 트리거 (single-shot OK) — "집이야"/"사무실이야"/"출장중이야" → dev-environments push (시크릿 마스킹 검증 fail 시 자동 cancel)

금지 .env, credential, 대용량 바이너리, 민감 파일 commit
