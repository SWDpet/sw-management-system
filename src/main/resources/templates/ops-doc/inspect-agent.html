점검 수집모듈 다운로드

목록으로
inspect-agent v0.0.0

지자체 서버(AP/DB/GIS)에서 점검 메트릭을 수집하는 Windows PowerShell 모듈입니다.

빌드 시각
-
파일 수
-
SHA-256
-
다운로드 (.zip)
수집모듈 배포본이 아직 준비되지 않았습니다. (빌드 패키징 누락 — 관리자 문의)

설치 / 실행 요약

  1. 다운로드한 zip 을 점검 대상 서버에 압축 해제 (inspect-agent/ 폴더 생성)
  2. setup.ps1 (또는 setup-gui.ps1) 실행 — 대화형 설정 wizard
  3. config/site.example.json 을 site.<코드>.json 으로 복사 후 현장 값 입력
  4. inspect.ps1 실행 → QR 생성 → SW Manager 점검내역서에 업로드

상세 절차는 압축 내 README.md 를 참고하세요. 무결성 검증은 위 SHA-256 값과 다운로드 파일의 해시를 대조하세요.
